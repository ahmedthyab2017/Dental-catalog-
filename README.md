# 🦷 Dental Clinic Manager + Gallery SaaS

**Dual SaaS Platform** for modern dental clinics

## 📊 Products

### 1. Clinic Manager Pro ($15-29/month)
- Patient file management
- Session tracking with timeline
- Treatment types & costs
- Payment tracking (paid/pending)
- Built-in design templates
- 4 before + 4 after photos free

### 2. Dental Gallery (Free + Premium)
- Free: 4 before + 4 after photos
- Premium: Unlimited photos + templates
- Social media ready designs
- Auto-generate marketing content

## 🛠 Tech Stack
- **Frontend**: Next.js 14 + React 18
- **Backend**: Next.js API Routes
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth + Google OAuth
- **Storage**: Supabase Storage + Cloudinary
- **Payments**: Stripe
- **Hosting**: Vercel
